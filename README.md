📝 О проекте
Одностраничное портфолио, созданное для демонстрации навыков, проектов и увлечений. Сайт выполнен в минималистичном стиле с поддержкой трёх цветовых тем: тёмной, светлой и «терминальной» зелёной.

Что можно увидеть на сайте:
Приветственный блок — краткое представление и специализация

Навыки — Python, Code Combat, Scratch, HTML, VS Code

Проекты — учебные и любительские наработки

Опыт — нестандартный взгляд на жизненный путь 😄

Хобби — гитара, рисование, видеоигры (со списком пройденных игр)

Контакты — email, телефон, Telegram, TikTok

🛠️ Используемые технологии
HTML5 — семантическая разметка

CSS3 — кастомные переменные для тем, адаптивная вёрстка, анимации

JavaScript — переключение тем (dark/light/green), загрузка аватарки через localStorage, раскрывающийся список игр

Font Awesome — иконки

Google Fonts — шрифт Inter

✨ Особенности
🎨 Три темы оформления — тёмная, светлая и зелёная «терминальная»

🖼️ Загрузка аватарки — изображение сохраняется в localStorage браузера

🎮 Интерактивный блок с играми — клик по карточке «Видеоигры» раскрывает список пройденных игр

📱 Адаптивный дизайн — корректно отображается на всех устройствах

🌙 Сохранение настроек — выбранная тема и аватарка запоминаются

🚀 Запуск локально
Сайт полностью статический, для запуска достаточно открыть index.html в браузере:

bash
git clone https://github.com/shadow4noreason/shadow4noreason.github.io.git
cd shadow4noreason.github.io
# Открыть index.html в браузере
Или использовать любой локальный сервер, например:

bash
python -m http.server 8000
# или
npx serve .
